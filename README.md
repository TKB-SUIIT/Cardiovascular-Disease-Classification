# Cardiovascular-Disease-Classification
The Machine Learning based python programming for Cardiovascular Disease Classification includes 20 classical ML algorithms from each house, compares the results of this models using performance measures. Dataset link: https://www.kaggle.com/datasets/sid321axn/heart-statlog-cleveland-hungary-final/data. Publication: https://doi.org/10.1080/10543406.2024.2429524
